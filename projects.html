Zero Trust Cloud-Based Passwordless MFA Framework

Python | Docker | Keycloak | Cloudflare Tunnel | AWS

Degree Final Year Project: This was my final year degree project. I designed and built a highly secure login system that removes the need for passwords. Instead, it uses passkeys and biometric verification like fingerprints or face scans to confirm who you are, based on Zero Trust security principles.
